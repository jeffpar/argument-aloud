## USSC Advocates

The [Advocates Data](advocates.json) is generated by the [update_advocates](../../../scripts/update_advocates.py) script.

The [Women Advocates CSV](../../../data/courts/ussc/women.csv) comes from the [Lone Dissent](https://github.com/jeffpar/lonedissent) project.  Since that project is no longer being updated, it will be updated here, through periodic runs of the `update_advocates` script.

NOTE: The *definitive* source of information about Women Advocates comes from Marlene Trestman.  See her article "[Women Advocates Before the Supreme Court](https://supremecourthistory.org/oral-arguments/women-advocates-before-the-supreme-court/)", which links to a [table](https://docs.google.com/spreadsheets/d/1Qsu5_yl8WABum3OmNqi8wruhWXeC73n9/edit?usp=sharing&ouid=109229861586313228368&rtpof=true&sd=true) that we have downloaded [here](../../../data/courts/ussc/Women%20Advocates%20Through%20October%20Term%202024.csv).
